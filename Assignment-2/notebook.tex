\section{The Iris Dataset}\label{the-iris-dataset}}

This data sets consists of 3 different types of irises' (Setosa,
Versicolour, and Virginica) petal and sepal length, stored in a 150x4
numpy.ndarray

The rows being the samples and the columns being: Sepal Length, Sepal
Width, Petal Length and Petal Width. (Sepal: 萼片, Petal: 花瓣, length
and width is in centimeters）

The below plot uses the first two features. See
\texttt{here\ \textless{}https://en.wikipedia.org/wiki/Iris\_flower\_data\_set\textgreater{}}\_
for more information on this dataset.
